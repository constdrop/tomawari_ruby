# 遠回りRuby

![tomawari_ruby](tomawari_ruby_256.png)

## 概要

遠回りRubyは、Markdownで作成している、Rubyを学習するための教材です。

## 特長

* Rubyの公式ページの「Rubyとは」を解説してみました。
* 型、配列、ハッシュなど、データの基本について丁寧に書きました。
* オブジェクト指向について丁寧に書きました。

## 要件

以下の環境での使用を想定しています。
* Ruby 3.3
* Windows 10/11
* VSCode

## 使い方

Markdownで作成していますので、
* GitHubでそのまま表示できます。
* PDFへの変換や、印刷して読むことも可能です。
